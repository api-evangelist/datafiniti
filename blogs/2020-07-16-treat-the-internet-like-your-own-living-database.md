<p><em>Introducing pagination for the Datafiniti API</em></p><p>Today Datafiniti is adding a brand new feature to our API — the ability to paginate through search results from any of our databases in real-time. Previously, if you had made an API call through Datafiniti, you were limited to either (a) a maximum of the 1,000 most relevant search results, or (b) downloading a data set in bulk through a batch process. With the introduction of pagination, you can now scroll through as many results as you want in real-time.</p><h3>How to Use Pagination</h3><p>Using pagination on the API is easy. Let’s say you initially run a search like:</p><p>POST <a href="https://api.datafiniti.co/v4/products">https://api.datafiniti.co/v4/products</a></p><p>{</p><p>“query”: “brand:Apple”,</p><p>“format”: “json”</p><p>}</p><p>This would give you the 10-most relevant results for Apple-branded products. If you want to then get the next 10-most relevant results, you just need to do this:</p><p>POST <a href="https://api.datafiniti.co/v4/products/paginate?page=2&amp;limit=10">https://api.datafiniti.co/v4/products/paginate?page=2&amp;limit=10</a></p><p>{</p><p>“query”: “brand:Apple”,</p><p>“format”: “json”</p><p>}</p><p>You can check out more detailed documentation here:</p><ul><li><a href="https://developer.datafiniti.co/docs/constructing-business-queries#section-paging-over-results">Paginating through business data</a></li><li><a href="https://developer.datafiniti.co/docs/constructing-people-queries#section-paging-over-results">Paginating through people data</a></li><li><a href="https://developer.datafiniti.co/docs/constructing-product-queries#section-paging-over-results">Paginating through product data</a></li><li><a href="https://developer.datafiniti.co/docs/constructing-property-queries#section-paging-over-results">Paginating through property data</a></li></ul><h3>A Living Database for the Entire Internet</h3><p>With the introduction of pagination, you can now create a live connection between your application and Datafiniti. As an example, if you have a search feature that lets users scroll through data on businesses, people, products, or property, you can run that search directly through our API. To your user, it will seem like their browsing through the entire Internet directly from your application.</p><p>Want to try out this new feature for yourself?
