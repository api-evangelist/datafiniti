# Datafiniti (datafiniti)
Datafiniti is a Data as a Service (DaaS) provider that collects, organizes, and standardizes large-scale data from the public web, delivering ready-to-use datasets for property, people, business, and product data through their API, web portal, and bulk downloads.

## APIs

### Datafiniti API
Unified Datafiniti REST API exposing search and download endpoints for businesses, products, properties, and people datasets.

**Human URL:** [https://docs.datafiniti.co/docs/api-introduction](https://docs.datafiniti.co/docs/api-introduction)

### Datafiniti Business Data API
Business listings aggregated from hundreds of online directories and review websites with firmographics and reviews. Over 131 million business records available.

### Datafiniti Product Data API
Millions of product records spanning major retailers and brands. Over 506 million product records available.

### Datafiniti Property Data API
Real estate listings from dozens of websites with pricing, amenities, and reviews. Over 205 million property records available.

### Datafiniti People Data API
People data records aggregated from public web sources.
